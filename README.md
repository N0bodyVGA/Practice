## Формулировка вариативной части

В рамках вариативной части выбрана тема из репозитория
[build-your-own-x](https://github.com/codecrafters-io/build-your-own-x):

**«Собственный HTTP-сервер на Python»**.

Запланировано:
- исследование принципов работы HTTP и сокетов;
- разработка минимального HTTP-сервера «с нуля»;
- подготовка технического руководства для начинающих;
- документирование архитектуры, процессов и примеров кода в Markdown.

## Быстрый старт

### Запуск сайта (без Hugo)

```bash
python -m http.server 5500 --directory site
```

После запуска сайт будет доступен по адресу `http://localhost:5500/index.html`.
