Вариативная часть: презентационный HTTP-сервер

Тема: «Собственный HTTP-сервер на Python без веб-фреймворков». Сервер реализован через TCP-сокеты и оформлен как небольшое интерактивное веб-приложение.

Что сделано

  • Добавлена современная HTML-страница с CSS-оформлением в стиле PolyQuest.
  • Реализованы маршруты /, /about, /routes, /api/status, /hello?name=... и /health.
  • Появились JSON API, динамическая страница приветствия, красивая 404-страница и обработка метода 405.
  • Улучшены заголовки HTTP-ответа, логирование запросов и структура кода.

Как запустить

Из корня проекта выполните команду:

                                                python src/main.py

После запуска откройте в браузере:

                                                    http://127.0.0.1:8080/
